# Documentation Index

当前文档按范围拆分，避免把所有信息堆在一个文件里。

| Document | Scope |
| --- | --- |
| `ARCHITECTURE.md` | App 总体架构、provider、数据流、核心约束。 |
| `FILE_STRUCTURE.md` | 文件结构、每个文件/目录的作用和本地依赖。 |
| `UI_ROUTES.md` | 当前可用/可达 UI、路径、入口、重定向规则。 |
| `API_REFERENCE.md` | App 内部 API、Supabase tables/RPC、本地同步 API。 |
| `COMPONENTS.md` | 组件库、hooks、第三方库、可复用 helper。 |
| `DESIGN_SYSTEM.md` | 视觉 token、布局、交互规范。 |
| `DEVELOPMENT_NOTES.md` | 环境、离线同步、固定支出、路由和开发注意事项。 |
